# Atendimento IA — Templates por Nicho

Coleção de aplicativos web de atendimento com inteligência artificial para profissionais liberais — desenvolvidos em HTML, CSS e JavaScript puro, prontos para personalização por nicho.

## Sobre o projeto

Cada template é um sistema completo de atendimento ao cliente com interface profissional, dark/light mode e integração com IA via n8n para respostas automáticas baseadas no conhecimento do profissional (RAG).

## Templates disponíveis

### Psicólogo / Psicóloga
- Interface com identidade visual para clínica de psicologia
- Agendamento de consultas
- Atendimento receptivo com IA

### Dentista / Odontologia
- Interface com identidade visual para clínica odontológica
- Gestão de agendamentos
- Atendimento automatizado

### Advogado / Escritório Jurídico
- Interface com identidade visual para escritório de advocacia
- Triagem de casos
- Atendimento inicial automatizado

## Como funciona

Cada template se conecta a um agente de IA configurado via n8n, que responde com base em uma base de conhecimento personalizada do profissional — horários, especialidades, valores, perguntas frequentes.

## Stack

- HTML5
- CSS3 (com tokens de design, dark/light mode)
- JavaScript puro
- Integração via n8n (webhooks)
- Supabase (dados de sessão e configuração)

## Como personalizar

1. Abra o `index.html` do nicho desejado
2. Substitua nome, cores e informações do profissional
3. Configure o webhook do seu agente n8n
4. Hospede em qualquer servidor estático (Netlify, Vercel, GitHub Pages)
